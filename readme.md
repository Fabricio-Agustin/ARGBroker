# ARGBROKER

**ARGBROKER** es una aplicación de consola (CLI) diseñada para la gestión y simulación de un portfolio de inversiones en el mercado de acciones. 

El sistema permite interactuar mediante una interfaz de terminal para gestionar usuarios, simular transacciones financieras y mantener un control en tiempo real de los activos.

---

##  Características Principales

* **Gestión de Usuarios:** Sistema de registro e inicio de sesión (Login) seguro para resguardar la información de cada inversor.
* **Control de Portfolio:** Visualización clara del saldo disponible en efectivo, las acciones en posesión y el rendimiento actual de las inversiones.
* **Módulo de Transacciones:** * **Compra de Acciones:** Selección de tickers y cálculo automático de costos debitados del saldo.
    * **Venta de Acciones:** Liquidación de activos para la toma de ganancias con acreditación inmediata de fondos.

---

##  Tecnologías Utilizadas

* **Lenguaje:** Python
* **Base de datos:** Mysql
* **Herramientas:** Docker, Git, Github
